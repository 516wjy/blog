洛谷 At163 题解

2020-03-24

先来分析一下

  1. 我们先读入当前字符串
  2. 再反转当前字符串，然后将 nnn 个字符串按字典序排序
  3. 最后再次翻转并输出即可。

对于反转字符串，可以用reverse()轻松搞定。

#include<iostream>
#include<cstdio>
#include<cstring>
#include<algorithm>
using namespace std;
int main(){
    int n;
    cin>>n;
    string a[105];
    for(int i=0;i<n;i++){
        cin>>a[i];
        reverse(a[i].begin(),a[i].end());
    }
    sort(a,a+n);
    for(int i=0;i<n;i++){
        reverse(a[i].begin(),a[i].end());
        cout<<a[i]<<endl;
    }
    return 0;
}
