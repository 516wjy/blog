Archlinux QV2ray 实现科学上网

2020-06-27

1. 准备

  • 有一套机场订阅，并且包含V2ray。
  • 有一台装好的 ArchLinux， Manjaro 也行

2. 配置 Pacman Archlinuxcn

在 /etc/pacman.conf 的末尾添加这几行代码

[archlinuxcn]
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

导入 GDP-key

sudo pacman -Syy
sudo pacman -S archlinuxcn-keyring

3. 安装 Qv2ray

sudo pacman -S qv2ray

4. 下载 V2ray-core

  1. 在浏览器中访问 https://github.com/v2ray/v2ray-core/releases 并下载 v2ray-linux-64.zip
  2. 解压到 ~/ 下并重命名为 .v2ray

5. 配置 Qv2ray

  1. 打开 Qv2ray 并选择 首选项 找到 内核 并把 核心可执行文件路径 改为 那个文件夹路径/v2ray ，把 V2ray资源目录 改成 那个文件夹路径 （文件夹路径应该是 /home/用户名/.v2ray ）
  2. 回到 常规设置 ，把 Qv2ray代理 改成 使用系统代理
  3. 退出 首选项，进入 订阅， 添加订阅地址，并点击 更新订阅数据
  4. 进入 首选项 ，找到 自动连接分组 选为你的分组，把 配置 选为你想要的线路

写在最后的话

祝大家都能实现科学上网！
